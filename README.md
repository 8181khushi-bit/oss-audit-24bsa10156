## Chosen Software
Git (Open Source Version Control System)


## Project Description
This project is based on the study of open source software using Git. It includes a report explaining the origin, license, and importance of Git, along with practical shell scripts performed on a Linux system.

## Scripts Included

### 1. system_info.sh
This script displays system information such as user name, kernel version, uptime, OS details, and date.

### 2. package_check.sh
This script checks whether Git is installed in the system and shows its version.

### 3. disk_audit.sh
This script displays disk usage and permissions of important system directories.

### 4. log_analyzer.sh
This script reads a log file and counts the number of error messages.

### 5. manifesto.sh
This script takes user input and generates a simple open source manifesto.

## How to Run Scripts

Step 1: Give permission
```bash
chmod +x system_info.sh
chmod +x package_check.sh
chmod +x disk_audit.sh
chmod +x log_analyzer.sh
chmod +x manifesto.sh

Step 2: Run scripts
./system_info.sh
./package_check.sh
./disk_audit.sh
./log_analyzer.sh
